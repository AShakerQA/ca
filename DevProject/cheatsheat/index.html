Table Tags

Sometimes in our webpage we need to visualise our data in tabular format, for this we can use table tags in HTML

HTML tags Definition
<table> The Table Element: represents tabular data
<th> The Table Header Element: Defines a cell as a header of a group of table cells
<tr> The Table Row Element: Defines a row of cells in a table
<td> The Table Data Cell Element: Defines a cell of a table that contains data

Table Attributes

Table attributes in HTML are used to customise tables

Attributes Definition
Colspan An attribute that indicates for how many columns the cell extends (Default is 1)
rowspan An attribute that indicates for how many rows the cell extends (Default is 1)

Semntic HTML for tables

Semnantic elements clearly describes its meaning to both the browser and the developer

Semantic HTML tags Definition
<thead> The Table Head Element: Defines a set of rows defining the head of the columns of the table
<tbody> The Table Body Element: Encapsulates a set of table rows
<tfoot> The Table Foot Element: Defines a set of rows summarizing the columns of a table
